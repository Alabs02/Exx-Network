# [EXX NETWORK DAPP](https://exx-network-teal.vercel.app/)

Presale is an ICO platform developed on BSC Testnet. <br />

### Application Setup

```bash
npm install -S
```

<br />

### To Start Development Server

```bash
npm run dev --host
```

<br />

### TECH STACK

- React JS
- TypecScript
- Vite
- Redux Toolkit
- Wagmi
- Rainbow Kit
- Ether <br /> <br />
